To run this program, you should open the task1.ipynb file in juypter notebook in Anaconda and has already installed CV2 package.

Before you run code, you need to create 2 empty folders called 'frames' and 'result' and make 
sure there is a monkey_taskA.mov in the folder. After that, you could run the code and it will output
video after finishing running it.

The currents frames in folder 'frames' and 'result' is the frames of original frames of money and 
arrowed monkey. The final_video.mov is the output result video.
